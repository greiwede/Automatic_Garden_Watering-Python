# python-sprinkler

Python Sprinkler is a Django based application for managing and controlling sprinklers in your garden.

It is supposed to run on a Raspberry Pi.

<h2>Setup</h2>
Download & Install Django
Download ZIP
Extract into Folder
Console >> python manage.py runserver
